# 天气查询 Skill（Weather MCP）

## 何时使用

当用户消息中包含以下意图时使用本技能：

- 询问某地天气：今天天气、北京天气、上海下雨吗、纽约温度
- 出行/穿衣建议：明天要不要带伞、穿什么、冷不冷

## 使用方式

通过 MCP 工具 `get_weather` 查询，无需单独脚本。Agent 在对话中直接调用该工具即可。

**工具参数：**

- `city`：城市或地点名称，支持中英文（如 "北京"、"上海"、"Tokyo"、"New York"）。若结果不唯一，可加上国家或省份（如 "北京, 中国"）。

## 返回内容

- 地点（名称 + 省份/国家）
- 当前气温（摄氏度）
- 天气状况（晴/多云/雨/雪等，中文描述）
- 风速（km/h）

## 数据来源与限制

- 使用 [Open-Meteo](https://open-meteo.com/) 免费 API，无需 API Key。
- 仅提供当前天气，不包含多日预报；若用户要「明天/下周」天气，可说明当前仅支持实时天气并建议用户稍后再问或使用其他服务。
