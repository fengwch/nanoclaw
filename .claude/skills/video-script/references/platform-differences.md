# YouTube vs Bilibili Platform Differences

两个平台的视频描述共享核心内容（概述 + 要点列表 + 相关链接），但包装方式不同。

## 差异对照表

| 区别 | YouTube | Bilibili |
|------|---------|----------|
| Hashtags | 描述开头放 `#tag` | 不使用 |
| 章节时间戳 | 必须有，从 `00:00` 开始 | 不需要 |
| 个人推广 | 描述底部，用 `---` 分隔 | 不放在描述中 |
| 赞助商广告 | 不放 | 描述开头第一行 |
| 跨视频引用 | YouTube 链接 | Bilibili 链接 |
| 播放列表 | 底部附播放列表链接 | 不需要 |

## 标题写作规则

1. **简洁有力**：控制在 50 字符内（YouTube 推荐）
2. **包含关键词**：便于搜索发现
3. **制造好奇/价值感**：让人想点进来
4. **避免标题党**：真实反映内容
5. **提供备选**：至少给 2-3 个标题方案
6. **两平台可以不同**：Bilibili 和 YouTube 的标题可以针对各自受众微调

## 描述写作规则

1. 前两行最重要（折叠前可见），放核心信息
2. 正文内容（概述 + 要点列表 + 相关链接）两平台共用，只是包装不同
3. YouTube 描述更完整（含个人推广、播放列表、章节）
4. Bilibili 描述更简洁，赞助商信息放在最前面

## 章节时间戳 (YouTube only)

1. 必须从 `00:00` 开始（YouTube 要求）
2. 根据脚本结构估算时间点
3. 章节名简洁，不超过 30 字符
4. 提醒用户：时间戳需在视频剪辑完成后根据实际时长调整
